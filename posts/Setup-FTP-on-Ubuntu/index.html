Setup FTP Service

Step 1. Install vsftpd

Well, everything starts with this tool.

Step 2. Configure vsftpd

The configuration file for vsftpd is located /etc/vsftpd.conf, you can refer to VSFTPD.CONF for all available options.

You should have sudo permission to edit it. And you can use / in Vim to search for options more efficiently.

Enable FTP Access

For security reason, we should only allow local users to access FTP.

Enable Uploads

Locate and uncomment the write_enable directive to allow filesystem changes, such as uploading and removing files.
