Procedures

Step 1. Prerequisites

Well, to deploy MySQL on a Linux server, you should have a Linux server first. Of course.

Step 2. Install MySQL

Just execute a ordinary install command.

Then, you can check the installation.

Step 4. Change Configuration

I’m not sure if this is needed, but I guess its somewhat useful. Just comment bind-address = 127.0.0.1 out, or change it to 0.0.0.0.

Then, restart MySQL service.

Step 3. Allow Access

Emm… One last step. Make sure the 3306 port is open. This can be found in the control panel of your server’s provider.

Now, we can use admin to connect to our server remotely.

Step 4. Connect MySQL with Navicat

In Navicat, crate a new connection. And just fill it with the host, username and password. And you are good to go.
